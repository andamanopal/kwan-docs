## Overview

Alto Niagara is our implementation of the Niagara Framework, providing the foundation for building automation and control. It handles device communication, basic automation logic, and user interface components.

## Key Features

<CardGroup cols={2}>
  <Card title="Device Communication" icon="network-wired">
    Native support for BACnet, Modbus, and other industrial protocols
  </Card>
  <Card title="Basic Automation" icon="gears">
    Simple control logic, alarms, and scheduling capabilities
  </Card>
  <Card title="User Interface" icon="desktop">
    3D visualization of chiller plant and real-time monitoring
  </Card>
  <Card title="Integration Layer" icon="plug">
    Seamless integration with Volttron and other Alto components
  </Card>
</CardGroup>

## Architecture

The Alto Niagara implementation follows a modular architecture:

<Steps>
  <Step title="Device Layer">
    Handles direct communication with physical devices and sensors
  </Step>
  <Step title="Control Layer">
    Implements basic automation logic and scheduling
  </Step>
  <Step title="Visualization Layer">
    Provides user interface and monitoring capabilities
  </Step>
  <Step title="Integration Layer">
    Manages communication with other Alto components
  </Step>
</Steps>

## Getting Started

To work with Alto Niagara, you'll need to:

1. Set up the Niagara Framework environment
2. Install the Alto Niagara module
3. Configure device connections
4. Set up basic automation rules

For detailed setup instructions, refer to the [Folder Structure](/components/alto-niagara/folder-structure) guide.
