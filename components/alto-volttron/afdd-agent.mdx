## Overview

The AFDD (Automated Fault Detection and Diagnostics) Agent continuously monitors system performance to identify potential issues and anomalies in building equipment operation. It uses advanced analytics and machine learning to detect faults early and provide diagnostic information.

## Usage Examples

### Monitoring Setup

```python
# Configure equipment monitoring
agent.configure_monitoring(
    equipment_id="chiller_01",
    parameters=["COP", "approach_temp", "power_consumption"],
    thresholds={
        "COP_min": 4.0,
        "approach_temp_max": 5.0
    }
)
```

### Fault Analysis

```python
# Analyze equipment performance
results = agent.analyze_performance(
    equipment_id="chiller_01",
    time_range={
        "start": "2024-01-16T00:00:00Z",
        "end": "2024-01-17T00:00:00Z"
    }
)
```
