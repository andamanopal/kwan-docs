## Overview

The ChillerSequence Agent optimizes the operation of multiple chillers by determining the most efficient sequence of operation. It considers factors such as load demand, energy efficiency, equipment wear, and operational constraints to make intelligent decisions about chiller staging.

## Configuration

### Basic Setup

```python
{
  "chillers": [
    {
      "id": "chiller_01",
      "capacity": 500,
      "min_load": 20,
      "max_load": 100,
      "efficiency_curve": "default_centrifugal",
      "priority": 1
    },
    {
      "id": "chiller_02",
      "capacity": 500,
      "min_load": 20,
      "max_load": 100,
      "efficiency_curve": "default_centrifugal",
      "priority": 2
    }
  ],
  "control": {
    "staging": {
      "method": "efficiency_based",
      "min_runtime": 1800,
      "min_offtime": 900,
      "stage_up_threshold": 85,
      "stage_down_threshold": 40
    },
    "optimization": {
      "target": "min_power",
      "constraints": {
        "max_starts_per_hour": 4,
        "min_load_per_chiller": 20
      }
    }
  }
}
```

## Usage Examples

### Sequence Configuration

```python
# Configure sequencing strategy
agent.configure_sequence(
    strategy="efficiency_based",
    parameters={
        "efficiency_weight": 0.7,
        "runtime_weight": 0.3,
        "min_stage_interval": 1800
    }
)
```

### Runtime Management

```python
# Set runtime constraints
agent.set_runtime_constraints(
    chiller_id="chiller_01",
    constraints={
        "max_daily_runtime": 18,
        "min_cycle_time": 1800,
        "max_starts_per_day": 6
    }
)
```
