## Overview

The TimescaleDB Agent is responsible for managing time-series data within the Alto Volttron platform. It handles data storage, retrieval, and management operations for sensor readings, control signals, and other time-series data.

## Configuration

### Basic Setup

```python
{
  "connection": {
    "host": "localhost",
    "port": 5432,
    "database": "alto_timescale",
    "user": "alto_user",
    "password": "your_password"
  },
  "tables": {
    "sensor_data": {
      "retention_days": 365,
      "chunk_interval": "1 day"
    },
    "control_signals": {
      "retention_days": 90,
      "chunk_interval": "1 hour"
    }
  }
}
```

## Usage Examples

### Writing Data

```python
agent.write_point(
    table="sensor_data",
    tags={
        "device_id": "chiller_01",
        "sensor_type": "temperature"
    },
    fields={
        "value": 23.5,
        "unit": "celsius"
    },
    timestamp="2024-01-17T10:00:00Z"
)
```

### Querying Data

```python
results = agent.query_data(
    table="sensor_data",
    start_time="2024-01-16T00:00:00Z",
    end_time="2024-01-17T00:00:00Z",
    tags={
        "device_id": "chiller_01"
    },
    aggregation="avg",
    interval="1h"
)
```
