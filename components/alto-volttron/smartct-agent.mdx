## Overview

The SmartCT (Smart Cooling Tower) Agent provides intelligent control and optimization of cooling tower operations. It manages fan speeds, water flow rates, and temperature setpoints to maximize cooling efficiency while minimizing energy consumption and water usage.

## Configuration

### Basic Setup

```python
{
  "cooling_tower": {
    "id": "ct_01",
    "type": "counter_flow",
    "capacity": {
      "heat_rejection": 2000,
      "units": "tons"
    },
    "fans": [
      {
        "id": "fan_01",
        "type": "variable_speed",
        "speed_limits": {
          "min": 20,
          "max": 100
        }
      }
    ]
  },
  "control": {
    "temperature": {
      "setpoint": {
        "value": 29.4,
        "units": "C",
        "reset": {
          "enabled": true,
          "min": 23.9,
          "max": 32.2,
          "wetbulb_approach": 2.8
        }
      }
    },
    "fan": {
      "staging": {
        "method": "temperature_based",
        "deadband": 1.1
      }
    }
  }
}
```

## Usage Examples

### Temperature Control Setup

```python
# Configure temperature control
agent.configure_temperature_control(
    parameters={
        "setpoint": 29.4,
        "reset_enabled": true,
        "reset_parameters": {
            "min_setpoint": 23.9,
            "max_setpoint": 32.2,
            "wetbulb_approach": 2.8,
            "evaluation_interval": 300
        }
    }
)
```

### Fan Control Setup

```python
# Configure fan control
agent.configure_fan_control(
    fan_id="fan_01",
    parameters={
        "control_method": "temperature_based",
        "speed_limits": {
            "min": 20,
            "max": 100
        },
        "staging": {
            "temperature_deadband": 1.1,
            "minimum_runtime": 300
        }
    }
)
```
