## Overview

The ChillerAddSubtract Agent manages the dynamic addition and removal of chillers from operation based on system load requirements. It optimizes the number of active chillers to match cooling demand while maintaining system efficiency and equipment longevity.

## Usage Examples

### Staging Configuration

```python
# Configure staging parameters
agent.configure_staging(
    parameters={
        "add_threshold": 85,
        "subtract_threshold": 45,
        "minimum_runtime": 1800,
        "evaluation_period": 600,
        "staging_delay": 300
    }
)
```

### Rotation Setup

```python
# Configure chiller rotation
agent.configure_rotation(
    parameters={
        "enabled": true,
        "interval_hours": 168,
        "evaluation_time": "00:00",
        "exclude_chillers": ["chiller_03"]
    }
)
```
