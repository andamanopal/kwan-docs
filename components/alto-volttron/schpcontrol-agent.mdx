## Overview

The SCHPControl (Secondary Chilled Water Pump Control) Agent optimizes the operation of secondary chilled water pumps in the HVAC system. It manages pump speeds, flow rates, and pressure differentials to maintain efficient distribution of chilled water while minimizing energy consumption.

## Configuration

### Basic Setup

```python
{
  "pumps": [
    {
      "id": "schp_01",
      "type": "variable_speed",
      "capacity": {
        "flow_rate": 1000,
        "units": "GPM"
      },
      "speed_limits": {
        "min": 20,
        "max": 100
      }
    }
  ],
  "control": {
    "pressure": {
      "setpoint": 15,
      "units": "PSI",
      "deadband": 1,
      "reset": {
        "enabled": true,
        "min_setpoint": 10,
        "max_setpoint": 20
      }
    },
    "flow": {
      "min_flow": 100,
      "units": "GPM",
      "optimization": {
        "enabled": true,
        "method": "load_based"
      }
    }
  }
}
```

## Usage Examples

### Pressure Control Setup

```python
# Configure pressure control
agent.configure_pressure_control(
    pump_id="schp_01",
    parameters={
        "setpoint": 15,
        "deadband": 1,
        "reset_enabled": true,
        "reset_parameters": {
            "min_setpoint": 10,
            "max_setpoint": 20,
            "reset_interval": 300
        }
    }
)
```

### Flow Management

```python
# Configure flow management
agent.configure_flow_control(
    pump_id="schp_01",
    parameters={
        "min_flow": 100,
        "max_flow": 1000,
        "optimization": {
            "method": "load_based",
            "parameters": {
                "load_factor": 0.8,
                "response_time": 120
            }
        }
    }
)
```
