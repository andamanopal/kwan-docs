## Overview

The ChillerSchedule Agent manages the operational schedules of chillers in the system. It coordinates start/stop timing, maintenance windows, and operational periods based on building occupancy, demand forecasts, and energy optimization goals.

## Usage Examples

### Schedule Configuration

```python
# Set up regular operation schedule
agent.configure_schedule(
    schedule_type="regular",
    parameters={
        "days": ["monday", "tuesday", "wednesday", "thursday", "friday"],
        "start_time": "06:00",
        "end_time": "19:00",
        "pre_cooling": {
            "enabled": true,
            "duration_minutes": 60
        }
    }
)
```

### Maintenance Planning

```python
# Schedule maintenance window
agent.schedule_maintenance(
    chiller_id="chiller_01",
    window={
        "start": "2024-02-01T08:00:00Z",
        "duration_hours": 4,
        "type": "preventive",
        "description": "Annual maintenance"
    }
)
```
