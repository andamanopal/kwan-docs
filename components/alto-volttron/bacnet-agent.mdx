## Overview

The BACnet Agent enables communication between Alto Volttron and BACnet-enabled devices. It handles device discovery, data polling, and command execution for building automation equipment using the BACnet protocol.

## Configuration

### Basic Setup

```python
{
  "device_scan_interval": 300,
  "default_proxy_address": "192.168.1.100",
  "networks": [
    {
      "network": 1234,
      "address": "192.168.1.0/24",
      "device_id_range": {
        "min": 1,
        "max": 4194303

    }
  ],
  "points": {
    "poll_interval": 300,
    "groups": [
      {
        "name": "critical",
        "interval": 60,

  }

```

## Usage Examples

### Device Discovery

```python
# Start device discovery
agent.discover_devices(
    network_number=1234,
    address_range="192.168.1.0/24"
)

# Get discovered devices
devices = agent.get_devices()
```

### Reading Points

```python
# Read a single point
value = agent.read_point(
    device_id=123,
    object_type="analogValue",
    object_instance=1
)

# Read multiple points
values = agent.read_points([
    {"device_id": 123, "object_type": "analogValue", "instance": 1},
    {"device_id": 123, "object_type": "binaryValue", "instance": 2}
])
```
